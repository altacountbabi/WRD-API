# Requirements:

1. A brain
2. Node.js and npm

# How to use:

1. Clone this repository
2. Open cloned repository with a terminal
3. Run this command in the terminal to install required dependencies: `npm i`
4. Run this command to start the dev server: `npm run dev`
5. Open http://localhost:5173 in your browser

# Known Issues:

1. This will only work on https://wrd-api.vercel.app/ for now, the folders that cause this issue are: /src/lib/routes/api/profileData and /src/lib/routes/api/threadData, please edit it if you want it to work on your own domain
